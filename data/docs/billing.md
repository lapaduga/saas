# Тарифы и оплата CloudApp

## Обзор тарифов

CloudApp предлагает четыре тарифных плана для удовлетворения потребностей различных пользователей — от индивидуальных до крупных корпораций.

### Free (Бесплатный)

**Цена:** $0/месяц

**Включено:**

- 5 ГБ облачного хранилища
- 1000 API-запросов в месяц
- Максимальный размер файла: 500 МБ
- До 3 рабочих пространств (workspace)
- Базовый шаринг ссылками
- 14 дней хранения удалённых файлов

**Ограничения:**

- Нет SSO-интеграции
- Нет вебхуков
- Только email-поддержка
- Нет расширенной аналитики
- Рекламные баннеры в интерфейсе

### Basic

**Цена:** $9/месяц (оплата ежемесячно) или $90/год (скидка 17%)

**Включено:**

- 50 ГБ облачного хранилища
- 10 000 API-запросов в месяц
- Rate Limit: 100 запросов в минуту
- Максимальный размер файла: 2 ГБ
- До 10 рабочих пространств
- Расширенные права доступа
- 30 дней хранения удалённых файлов
- Приоритетная email-поддержка

**Дополнительно:**

- История версий файлов (30 дней)
- Базовая аналитика использования
- Экспорт данных в CSV

### Pro

**Цена:** $29/месяц (оплата ежемесячно) или $290/год (скидка 17%)

**Включено:**

- 200 ГБ облачного хранилища
- 100 000 API-запросов в месяц
- Rate Limit: 500 запросов в минуту
- Максимальный размер файла: 10 ГБ
- Безлимитное количество рабочих пространств
- SSO-интеграция (Okta, Azure AD, Google)
- Вебхуки и автоматизации
- 90 дней хранения удалённых файлов
- Приоритетная поддержка 24/7
- Расширенная аналитика
- API для администрирования

**Дополнительно:**

- История версий файлов (1 год)
- Аудит активности
- Кастомные домены
- Белая метка (white-label)

### Enterprise

**Цена:** Договорная (от $99/месяц за пользователя)

**Включено:**

- Безлимитное облачное хранилище
- Безлимитные API-запросы
- Rate Limit: настраивается индивидуально
- Максимальный размер файла: 100 ГБ
- Безлимитное количество рабочих пространств
- Выделенная инфраструктура (опционально)
- SLA 99.99% uptime
- Персональный менеджер поддержки
- Кастомные интеграции
- Расширенное шифрование (CSE)

**Дополнительно:**

- Без ограничений по хранению удалённых файлов
- Расширенный аудит с экспортом
- LDAP-интеграция
- Кастомные политики безопасности
- Тренировки для команды
- Технический архитектор

## Управление подпиской

### Смена тарифа

1. Перейдите в **Настройки → Подписка**
2. Нажмите «Изменить план»
3. Выберите новый тариф
4. Подтвердите изменения

**При повышении тарифа:**

- Разница рассчитывается пропорционально оставшимся дням текущего периода
- Новые лимиты действуют немедленно

**При понижении тарифа:**

- Изменения вступают в силу с начала следующего биллингового периода
- Текущие лимиты сохраняются до конца периода
- Данные сверх нового лимита не удаляются, но новые загрузки блокируются

### Остановка подписки

Подписку можно приостановить на срок до 90 дней:

- Данные сохраняются и доступны для скачивания
- API-запросы блокируются
- Новая загрузка невозможна
- Плата не взимается

Для возобновления активируйте подписку в любое время.

### Отмена подписки

1. Перейдите в **Настройки → Подписка → Отменить**
2. Подтвердите отмену

**После отмены:**

- Подписка активна до конца текущего периода
- Данные хранятся 30 дней после окончания периода
- API-ключи деактивируются
- Через 30 дней аккаунт переводится на Free-план
- Данные сверх лимита Free удаляются через 7 дней

## Способы оплаты

### Банковские карты

- Visa, MasterCard, МИР
- American Express (только Pro/Enterprise)
- Дебетовые и кредитные карты

### Корпоративные счета

Доступно для Enterprise-клиентов:

- Оплата по счёту (invoice) с отсрочкой 30 дней
- Поддержкаultiple currencies
- Автоматическое создание инвойсов

### Промокоды

Промокоды применяются в **Настройки → Подписка → Промокод**:

- Скидка на первый платёж
- Скидка на продление
- Бесплатный период

## Лимиты и превышения

### API-лимиты

При превышении лимита API-запросов сервер возвращает:

```json
{
  "error": "rate_limit_exceeded",
  "message": "Monthly API limit reached. Upgrade your plan.",
  "retry_after": null,
  "current_usage": 10000,
  "limit": 10000
}
```

### Лимиты хранилища

При заполнении хранилища:

- Новая загрузка блокируется
- Существующие файлы остаются доступными
- Появляется уведомление в интерфейсе
- Рекомендуется удалить ненужные файлы или повысить тариф

### Rate Limiting (ограничение частоты)

Rate limit устанавливается по окну в 1 минуту:

```http
HTTP/1.1 429 Too Many Requests
X-RateLimit-Limit: 500
X-RateLimit-Remaining: 0
X-RateLimit-Reset: 1721155260
Retry-After: 45
```

Рекомендуется реализовать exponential backoff при получении 429.

## Возврат средств

### Политика возврата

- Возврат в течение 14 дней после оплаты — 100%
- Возврат в течение 30 дней — 50%
- После 30 дней — возврат не производится

### Процесс возврата

1. Напишите на billing@cloudapp.com с указанием:
   - Email аккаунта
   - Дата и сумма платежа
   - Причина возврата
2. Рассмотрение занимает до 5 рабочих дней
3. Средства возвращаются на ту же карту в течение 10 рабочих дней

## Налоги и инвойсы

### Налоги

- Цены указаны без НДС
- НДС добавляется в соответствии с законодательством вашей страны
- Для юридических лиц ЕС доступен reverse charge

### Инвойсы

Автоматические инвойсы генерируются в **Настройки → Подписка → История платежей**:

- PDF-инвойсы с реквизитами
- Экспорт в CSV для бухгалтерии
- Поддержка multiple billing contacts
