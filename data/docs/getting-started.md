# Быстрый старт с CloudApp

## Добро пожаловать в CloudApp

CloudApp — это современная облачная платформа для хранения, управления и обработки файлов с мощным REST API. Платформа предназначена для команд и индивидуальных пользователей, которым нужен надёжный и безопасный облачный сервис.

## Регистрация аккаунта

### Шаг 1: Создание аккаунта

Перейдите на страницу registration.cloudapp.com и заполните форму регистрации:

- **Email** — рабочий email для подтверждения
- **Имя** — отображаемое имя пользователя
- **Пароль** — минимум 8 символов, одна заглавная буква и одна цифра
- **Компания** — опциональное поле для бизнес-аккаунтов

После регистрации на указанный email придёт письмо с ссылкой подтверждения. Кликните по ссылке для активации аккаунта.

### Шаг 2: Первый вход

После подтверждения email войдите в систему, используя email и пароль. При первом входе откроется мастер настройки, который поможет:

1. Выбрать тарифный план
2. Настроить профиль
3. Загрузить первый файл
4. Пригласить коллег

## Загрузка файлов

### Через веб-интерфейс

1. Откройте раздел «Файлы» в боковом меню
2. Нажмите кнопку «Загрузить» или перетащите файлы в область загрузки
3. Выберите destination-папку (опционально)
4. Дождитесь завершения загрузки

Поддерживаются все распространённые форматы файлов: документы, изображения, видео, архивы, исходный код.

### Через API

```bash
curl -X POST https://api.cloudapp.com/v1/files \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf" \
  -F "path=/projects/"
```

Ответ сервера:

```json
{
  "id": "file_abc123",
  "name": "document.pdf",
  "size": 1048576,
  "path": "/projects/",
  "url": "https://storage.cloudapp.com/file_abc123",
  "created_at": "2026-07-16T10:00:00Z"
}
```

## Управление файлами

### Просмотр и навигация

Все файлы отображаются в виде списка или сетки. Используйте фильтры для поиска по:

- Типу файла
- Дате загрузки
- Размеру
- Владельцу

### Папки и структура

Создавайте папки для организации файлов. Поддерживается вложенность до 10 уровней. Для перемещения файла перетащите его в целевую папку.

### Права доступа

Для каждого файла или папки можно настроить:

- **Владелец** — полный контроль
- **Редактор** — чтение, запись, загрузка
- **Читатель** — только просмотр и скачивание
- **Нет доступа** — запрет на доступ

## Совместная работа

### Приглашение коллег

1. Перейдите в Настройки → Команда
2. Нажмите «Пригласить»
3. Введите email коллеги
4. Выберите роль: Редактор или Читатель
5. Отправьте приглашение

Коллега получит email-приглашение и сможет присоединиться к workspace.

### Комментарии и обсуждения

К каждому файлу можно оставлять комментарии. Используйте @упоминания для привлечения внимания конкретных коллег. Уведомления приходят на email и в настольное приложение.

## Настройка безопасности

### Рекомендации по безопасности

1. Включите двухфакторную аутентификацию (2FA)
2. Используйте сложные пароли (минимум 12 символов)
3. Регулярно ротируйте API-ключи
4. Настройте SSO для команды (Pro/Enterprise)
5. Проверяйте логи доступа в разделе «Безопасность»

### Аудит активности

В разделе «Настройки → Безопасность → Журнал» отображаются все действия в аккаунте:

- Входы в систему
- Загрузки и скачивания файлов
- Изменения прав доступа
- API-запросы
